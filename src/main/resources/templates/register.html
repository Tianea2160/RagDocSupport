Library Mode

doc-sources.yml

Pre-configured libraries with automatic URL resolution. Just select and enter a version.

Custom URL Mode

Register documentation from any URL. Useful for libraries not in doc-sources.yml or custom documentation pages.

The crawler will start from this URL and follow links within the same path scope.

Bulk Mode

Register multiple libraries at once. Each library must be pre-configured in doc-sources.yml. Runs in parallel.
